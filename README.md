# Book Analyzer Project (CS I)

## Description
This project is a Python-based Book Analyzer that:
-Downloads books from Project Gutenberg
-Cleans and processes text
-Removes stop words
-Analyzes word frequency
-Displays results using a bar chart
-Allows users to manage a small digital library

## Feautures
-Add books to library
-Remove books
-Update book URLs
-Analyze word frequency
-Visual visualization of results

## Technologies Used
-Python
-Requests library
-Regular Expressions (re)
-Collections (Counter)

## How to Run
1. Install dependencies:
pip install requests
2. Run the program:
python main.py
